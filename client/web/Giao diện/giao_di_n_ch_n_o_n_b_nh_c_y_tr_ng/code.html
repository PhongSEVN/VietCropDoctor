Dựa trên hình ảnh phân tích, tôi xác định cây lúa của bạn đang mắc Bệnh Đạo ôn (Rice Blast) với độ tin cậy cao (95%).

Tổng quan
Triệu chứng
Xử trí

Nguyên nhân: Nấm Pyricularia oryzae gây ra.

Triệu chứng quan sát được: Vết bệnh điển hình hình thoi (mắt én), rộng ở giữa và nhọn ở hai đầu, mép ngoài màu nâu đỏ, tâm vết bệnh màu xám tro. Các vết bệnh đang có xu hướng liên kết làm khô lá.

Khuyến nghị Xử trí Khẩn cấp:
  • Ngừng bón phân đạm (Ure) ngay lập tức.
  • Giữ mực nước ruộng ổn định (3-5cm).
  • Sử dụng thuốc trừ nấm đặc trị chứa hoạt chất Tricyclazole hoặc Isoprothiolane.
